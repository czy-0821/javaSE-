1.1 简答题一
简述一下集合和数组的区别？
答：
共同点：都可以存储数据
不同点：集合的容量是可变的，而数组的容量是不可变的。

1.4 简答题四
聊聊遍历集合的思路, 以及为什么要遍历集合, 并在下方编写出实际代码

答：
使用for循环，并通过size()方法获取到集合的长度来控制i的范围，然后通过get()方法和下标来获取集合中的值。

for (int i = 0; i < 集合名.size(); i++) {
            E date = 集合名.get(i);
        }


1.5 简答题五
集合中现有元素 ( "a" , "a" , "b" , "a" , "a") 如何能够将 "a" 字符串全部删除, 请说明思路, 并在下方
编写实际代码
答:

for (int i = 0; i < list.size(); i++) {
            String date = list.get(i);
            if(date.equals("a")){
                list.remove(i);
                i--;
            }
        }
