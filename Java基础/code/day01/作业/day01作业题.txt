1.4 简答题四
聊聊你对 Path 环境变量的看法, 解释下我们为什么要将 JDK 配置到 Path 当中



答：path环境变量可以使得程序在计算机的任何地方都可调用特定的程序，将jdk配置到path中就可以在其他地方编译运行java代码，不需要将程序写在jdk的bin目录下。

1.11 简答题十一
简述下什么是标识符, 在定义标识符的时候, 应该满足哪些规则和规范呢 ?




		标识符是用户编程时使用的名字，用与给类，方法，变量，常量命名。

		由字母、数字、下划线“_”、美元符号“$”组成
		第一个字符不能是数字，不能使用java中关键字作为标识符，区分大小写。
		变量名、方法名使用小驼峰命名法
		类名使用大驼峰命名法
		标识符要体现出其功能意义。
