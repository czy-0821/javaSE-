1.3 简答题三
简述下 ++ 运算符在参与运算使用的时候, 在前在后的区别

答：“++”在前的时候，先对数据进行加1操作，再进行其他操作；
“++”再后的时候，先对数据进行其他操作，再进行加1操作。


1.4 简答题四
现有代码如下, 请回答打印的结果, 并描述中间的运算过程

int a = 10;
int b = ++a;   
int c = a++;    
int result = b++ + ++c;
答:  23
“int b = ++a;”：先对a中保存的数据进行+1操作，b中保存的值为11，a中的值也改为11；
“int c = a++;”：先将a中的数据保存再c中，再对a中的数据进行+1操作，则c中保存这
11，a中保存着12；
“int result = b++ + ++c;”：c中数据+1后再与b中数据相加并保存再result中，然后b中数据+1。

1.9 简答题八
请结合下列代码, 描述 if 语句执行流程
if(判断条件1){
语句体1;
}else if(判断条件2){
语句体2;
}else if(判断条件3){
语句体3;
}else{
语句体n+1;
}
答:
先进行判断语句1的判断，true则执行语句体1，false则执行判断语句2，判断语句2结果为true则执行语句体2，判断语句2结果为false则执行判断语句3，判断语句3结果为true则执行语句3，判断语句3结果为false则执行语句体n+1。
