1.3 简答题三
请回答, 通过构造方法创建字符串对象, 对比双引号创建字符串对象, 两者有什么区别 ?

双引号创建字符串对象实在常量池中开辟空间，且内容相同的对象是共享一份数据的。

1.6 简答题六
需求 : 对字符串进行遍历操作, 可以使用下列的哪些方法 ? 具体怎么遍历呢	?

public boolean equals(String s)
public boolean equalsIgnoreCase(String anotherString)
public char[] toCharArray()
public char charAt(int index)
public int length()
public String replace(CharSequence target, CharSequence replacement)
public String substring(int beginIndex)
public String substring(int beginIndex, int endIndex)
public String[] split(String regex)

答:

1.toCharArray()  :将字符串转化为char类型的数组，然后遍历数组实现对字符串的遍历操作。

2.charAt()  :  通过索引值依次获取到字符串中不同位置的元素，从而实现对字符串的遍历操作。

3.substring（）：可以依次将字符串中每个位置的元素截取出来实现对字符串的遍历。

public boolean equals(String s)

public char[] toCharArray()

public int length()
public String replace(CharSequence target, CharSequence replacement)
public String substring(int beginIndex)
public String substring(int beginIndex, int endIndex)
public String[] split(String regex)
答:


先使用spilt("\\)")将字符串转变为  “（书名-价格”这种形式的数组，然后遍历数组，
对每一个对象使用spilt("-")，再次分为“（书名”，“价格”的这种形式，然后对书名使用substring方法

1.12 简答题十一
聊聊你对 StringBuilder 的理解, 什么时候你会想到使用 StringBuilder 呢 ?
答:

StringBuilder 是一个可变的字符串类，StringBuilder是一个可以一直向其中添加字符串对象的容器。

当我们需要不断的拼接字符串时，可以使用StringBuilder进行拼接，拼接结束后，再将其转变为String类型。
