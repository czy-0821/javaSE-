1.4 简答题四

答：

静态初始化:   类型 [] 数组名 = {数据,数据,数据......},     类型 [] 数组名 =new 类型[] {数据,数据,数据......}

动态初始化:   类型 [] 数组名 = new 类型[数组长度];

1.7 简答题七
请回答数组两种初始化有什么区别 ? 两种初始化各自适用于什么样的场景 ?

答：
静态初始化数组数据则一开始就由用户定义。适用于已经知道所有数据的值。

数组动态初始化就是只给定数组的长度，由系统给出默认初始化值。使用于一开始并不知道数据的值。
