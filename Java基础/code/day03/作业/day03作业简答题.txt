1.7 简答题七
现有如下代码 , 统计 1 ~ 20 之间, 5的倍数有几个, 但是代码存在错误, 请说明错误原因, 并更正代
码
public class Demo {
public static void main(String[] args) {
for (int i = 1; i <= 20; i++) {
int count = 0;
if(i % 5 == 0){
count++;
}
System.out.println(count);
}
}
}
答:

（1）count创建在for循环体内，每次循环count会从新创建覆盖原有数据
（2）打印输出在for循环体内，每次循环都会打印count的值

更改后：

public class Demo {
public static void main(String[] args) {

int count = 0;

for (int i = 1; i <= 20; i++) {

if(i % 5 == 0){
count++;
}
}
System.out.println(count);
}
}

1.9 简答题九
请说出 Java 的三种循环分别是什么 ? 以及各自的区别
答:

for循环：先判断条件是否符合再执行循环体，条件控制语句所控制的自增体归属与for循环，循环结束后就不能被访问到了
while循环：先判断条件是否符合再执行循环体，条件控制语句所控制的自增体不归属与for循环，循环结束后任然可以被访问到了

dowhile循环：先执行循环体再判断条件是否符合，条件控制语句所控制的自增体不归属与for循环，循环结束后任然可以被访问到了
